# Assets

## Purpose

This folder stores visual assets used by the public README and documentation. Assets should clarify TraceRail concepts without becoming a second source of workflow truth.

## Assets

| File | Used By | Purpose |
| --- | --- | --- |
| `tracerail-module-flow.gif` | `README.md` | Animated overview of a TraceRail rail, swappable modules, optional gates, typed outcomes, and durable memory. |

## Update Rule

When an asset changes, update the README or documentation that embeds it, preserve the source reason in the relevant feature package, and run the template checker before committing.
