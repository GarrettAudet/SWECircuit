# Memory Module

Preserve durable knowledge and retrieval pointers.

## Input

- Review outcome.
- Decisions.
- Durable lessons.
- Source artifacts.

Update active context, decisions, known issues, failed attempts, patterns, glossary, history ledger, and retrieval index when relevant.

- Memory entries.
- Retrieval pointers.

## Outcome

`pass` or `learn`.

## Artifacts

- `docs/memory/`
- Feature package.
- Milestone.
