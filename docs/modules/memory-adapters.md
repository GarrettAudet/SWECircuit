# Memory Adapters Module

## Purpose

Connect source-preserving memory to optional external memory systems.

## Input

- Source artifacts.
- Durable lessons.
- Retrieval pointers.
- Memory policy.

## Action

Preserve source truth, distill durable summaries, and keep future retrieval fast.

## Output

- Memory update.
- Retrieval pointer.
- Optional external memory record.

## Gate

Memory remains inspectable, source-preserving, private enough, and reversible, or adapter adoption routes to `deferred` or `rejected`.

## Outcome

`pass`, `learn`, `deferred`, or `rejected`.

## Artifacts

- `docs/memory/`
- History ledger.
- Retrieval index.
- Adapter evaluation.

## Adapter

Basic Memory, Mem0, Zep, Graphiti, A-MEM, and similar systems are optional memory adapters.
