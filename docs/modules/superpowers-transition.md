# Superpowers Transition Module

## Purpose

Move from user idea to readable design, approved plan, and implementation readiness.

## Input

- User idea.
- Current context.
- Existing docs or code.

## Action

Ask what the user is trying to do, tease out a spec, show design in readable chunks, plan implementation, and keep scope small.

## Output

- Clarified goal.
- Readable design.
- Implementation plan.
- Approval or assumptions.

## Gate

User or feature package approves the plan, or work routes to `clarify`.

## Outcome

`pass`, `clarify`, `split`, or `redesign`.

## Artifacts

- Spec.
- Plan.
- Tasks.
- IDE interaction notes.

## Adapter

Superpowers is an optional skills or plugin adapter. TraceRail extracts the methodology by default and installs nothing.
