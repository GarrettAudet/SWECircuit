# Root-Cause Analysis

Use this file when a failure required diagnosis or when a fix should teach future work.

Draft.

## Trigger

## Reproduction

What stable reproduction or evidence proved the failure?

## Confirmed Root Cause

State the confirmed cause. Include the evidence or experiment that confirmed it.

## Why It Was Missed

Explain why existing specs, tests, reviews, or assumptions did not catch it earlier.

## Fix

Describe the smallest causal fix that was applied.

## Regression Coverage

Describe the test or check that prevents this failure from returning.

## Follow-Up Work

List any work split out of scope.
