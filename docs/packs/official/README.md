# Official Packs

## Purpose

Official packs are curated optional bundles maintained by the TraceRail project. They let TraceRail package a repeatable capability without making it required core behavior.

## Current Official Packs

| Pack | Status | Purpose |
| --- | --- | --- |
| `tracepack-orchestration-readiness` | Official, not recommended yet | Prepare safe agent fan-out with decomposition, contracts, run records, verification, and memory. |

## Use Rule

Use an official pack when the core workflow is not enough for a repeated work shape, but the project does not need an external runtime or dependency. A pack can be copied, referenced, or adapted manually.

## Promotion Rule

Official does not mean recommended. A pack becomes recommended only after conformance evidence, dogfooding history, explicit permissions, rollback, and maintenance ownership are clear.
