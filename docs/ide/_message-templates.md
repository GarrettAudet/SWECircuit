# IDE Interaction Message Templates

Use these templates as concise chat shapes. Adapt wording naturally.

## Start Banner

```txt
Workflow:
Branch:
Stage:
Active artifact:
Next:
Assumptions:
```

## Stage Update

```txt
Stage:
Outcome:
Evidence:
Next:
```

## Assumption Notice

```txt
Assumption:
Why low-risk:
Recorded in:
Reversal path:
```

## Clarification Request

```txt
Decision needed:
Why it matters:
Options:
Recommendation:
Default if low-risk:
```

## Diagnosis Notice

```txt
Diagnosis mode:
Trigger:
Evidence captured:
Hypotheses:
Next experiment:
Stop condition:
```

## Verification Report

```txt
Verification:
Command:
Result:
Evidence location:
Skipped checks:
```

## Completion Handoff

```txt
Goal:
Outcome:
Files changed:
Verification:
Review state:
Memory updates:
Milestone:
Branch state:
Next gate:
```
